# cloudflare-ui
Command line user interface for cloudflare port forwarding site. 
# Installation
```git clone https://github.com/BHUTUU/cloudflare-ui```

```cd cloudfare-ui```

```bash config.sh```

# usage
```cloudflare --help```

```cloudflare http <port>```

# quick-install
